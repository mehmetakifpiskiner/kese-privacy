# Privacy Policy (Kese)

Last updated: 2026-03-05

Kese is a portfolio and personal finance tracking application. This policy explains what information we collect and how it is used.

## Information We Collect

### Account Information
- Email address (used for account creation and login via Supabase Auth).
- Password authentication is handled by Supabase Auth. We do not store plaintext passwords.

### Profile Information
- Full name provided by the user.

### Financial Data
- Portfolio information entered by the user.
- Debt records entered by the user.

## How We Use Information
The collected information is used to:
- Provide the core functionality of the app.
- Store and display portfolio and debt information.
- Maintain user accounts.

## Third Party Services
The app uses Supabase for authentication and database services.

## Data Security
Reasonable measures are taken to protect user data.

## Children’s Privacy
Kese is not intended for children under the age of 13.
