# Privacy Policy (Kese)

Last updated: 2026-03-05

Kese is a portfolio and personal finance tracking application. This policy explains what information we collect and how it is used.
For non-premium users, financial data may be stored locally on the user's device and not transmitted to external servers.

## Information We Collect

### Account Information
- Email address (used for account creation and login via Supabase Auth).
- Password authentication is handled by Supabase Auth. We do not store plaintext passwords.

### Profile Information
- Full name provided by the user.

### Financial Data
- Portfolio information entered by the user.
- Debt records entered by the user.

## How We Use Information
The collected information is used to:
- Provide the core functionality of the app.
- Store and display portfolio and debt information.
- Maintain user accounts.

## Third Party Services
The app uses Supabase for authentication and database services.

## Data Security
Reasonable measures are taken to protect user data.

## Children’s Privacy
Kese is not intended for children under the age of 13.

## Account Deletion

Users can delete portfolio and debt records directly within the application. 
Full account deletion can be requested by contacting kese.mobil@gmail.com.

To delete your account:

1. Send an email to: kese.mobil@gmail.com  
2. Use the subject: "Account Deletion Request"  
3. Include the email address used in the Kese app.

Once the request is received, the account and associated data will be permanently deleted within 7 days.

Deleted data includes:
- Full name
- Portfolio data
- Debt records

Authentication data is managed by Supabase Auth and will also be removed.
